# 🌿 Campus-Wide Sustainability Tracker
Comprehensive dashboard tracking campus energy, solar generation, water consumption, and carbon footprint.

## Quick Start
```bash
pip install -r requirements.txt && python data_generator.py && python model.py && python app.py
```
Dashboard: http://localhost:5009

## Features
- ☀️ **Solar Tracking** — Rooftop panel generation vs consumption
- 🌍 **CO₂ Analytics** — Emissions tracking and savings from solar
- 💧 **Water Monitoring** — Hourly consumption patterns
- 📊 **Sustainability Score** — Composite daily score with trend analysis
- 🤖 **Random Forest** — Energy demand prediction

## API: `/api/historical` · `/api/predict` · `/api/stats` · `/api/sustainability`

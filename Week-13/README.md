# 📊 Dashboard Visualization
Comprehensive multi-source campus analytics dashboard with 7 interactive chart types.

## Quick Start
```bash
pip install -r requirements.txt && python data_generator.py && python app.py
```
Dashboard: http://localhost:5012

## Features
- ⚡ **Energy Monitoring** — Campus-wide consumption trends
- 🌐 **Network Analytics** — Bandwidth usage patterns
- 🚶 **Footfall Tracking** — Campus traffic heatmap
- 🌬️ **Air Quality** — AQI monitoring with color-coded indicators
- 🎪 **Event Tracking** — Attendance visualization
- 📉 **Correlation Matrix** — Cross-source metric correlations

## API: `/api/data` · `/api/summary` · `/api/hourly` · `/api/daily`

# 🅿️ Parking Lot Lighting Forecast
Predict parking lot lighting energy using Decision Tree Regression with sunset/sunrise tracking.

## Quick Start
```bash
pip install -r requirements.txt && python data_generator.py && python model.py && python app.py
```
Dashboard: http://localhost:5007

## Features
- 🌅 **Sunset/Sunrise Tracking** — Dark vs light hour analytics
- 🌲 **Decision Tree** — Interpretable prediction model
- 💡 **Zone Lighting Analysis** — Floodlight, pathway, entry zones
- 🎉 **Event Detection** — Friday night event energy spikes

## API: `/api/historical` · `/api/predict` · `/api/stats` · `/api/lighting-analysis`

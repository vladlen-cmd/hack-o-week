# 🧺 Hostel Laundry Peak Prediction
Predict hostel laundry room energy peaks using KNN Regression with Sunday surge detection.

## Quick Start
```bash
pip install -r requirements.txt && python data_generator.py && python model.py && python app.py
```
Dashboard: http://localhost:5008

## Features
- 🔮 **KNN Regression** — Distance-weighted neighbor prediction
- 📅 **Sunday Surge** — Detect weekly laundry day peaks
- 🧺 **Machine Tracking** — Active washer/dryer correlation
- ⏰ **Peak Hour Detection** — Find optimal scheduling windows

## API: `/api/historical` · `/api/predict` · `/api/stats` · `/api/peak-analysis`

# 🔍 Basic Anomaly Detection
Multi-sensor anomaly detection using Isolation Forest and Z-Score with confusion matrix analysis.

## Quick Start
```bash
pip install -r requirements.txt && python data_generator.py && python detector.py && python app.py
```
Dashboard: http://localhost:5014

## Features
- 🌲 **Isolation Forest** — Unsupervised anomaly detection
- 📏 **Z-Score** — Statistical threshold-based detection
- 🎯 **Confusion Matrix** — Precision, Recall, F1 metrics
- ⚡ **Power Grid Monitoring** — Spike detection
- 🌡️ **Server Room Temp** — Cooling failure alerts
- 🌐 **Network Traffic** — Drop detection
- 📊 **Anomaly Score Distribution** — Score histogram
